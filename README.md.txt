# Smart Leave Manager

A Java Spring Boot application to manage employee leaves, leave approvals, and leave history.  
The system supports employee account creation, leave requests, approvals, and admin monitoring. Built with Spring Boot, Hibernate, and MySQL.

## âš¡ Installation & Usage
1. Clone the repository:
```bash
git clone https://github.com/Kddeshmukh/Smart-Leave-Manager.git
```
2. Open the project in IntelliJ IDEA or Eclipse.
3. Update `application.properties` with your MySQL database credentials.
4. Build the project using Maven:
```bash
mvn clean install
```
5. Run the Spring Boot application:
```bash
mvn spring-boot:run
```
6. Open your browser at:
```
http://localhost:8080/
```
